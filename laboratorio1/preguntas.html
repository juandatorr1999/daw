  • ¿Cuál es la diferencia entre Internet y la World Wide Web? La diferencia es que Internet es la red de computadoras, WWW es el modelo de intercambio de información basado en el protocolo HTTP/S(Veal,2010)
  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?Es para indicar la acción sobre el recurso especificado en el servidor(Puentes,2018)
  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio?Por el método POST ¿Por qué?Debido a que debes crear una entidad, sin embargo GET también lo puede hacer sin embargo, en el ejemplo debe ser POST en afán de ocultar los inputs privados como la contraseña y usuario(Puentes,2018)
  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?El método GET
  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? El código 200 significa que la petición ocurrió con éxito¿Ocurrió algún error?No
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404?Sí¿Por qué?Si son hipervínculos es responsabilidad del desarrollador, sin embargo si es un usuario que quiere crawlear el sitio debe ser bloqueado
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500?Sí ¿Por qué?El código significa que el servidor no pudo manejar una excepción
