Cuestionario

  • ¿Qué es entonces una base de datos - DB? Es un conjunto de datos relacionados, por datos se entiende que son hechos que pueden se almacenados y tienen un significado implicíto
  • ¿En qué casos es conveniente usar bases de datos?Cuando existan accesos concurrentes al mismo dato, cuando se necesita de permisos, cuando se quiera filtrar información deacuerdo a una regla, y finalmente cuando existan varios registros
  • ¿Qué es un sistema de gestión de base de datos - DBMS?Es el conjunto de datos relacionados y un conjunto de software que accede a ellos
  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Definir las estructura de la base de datos
    • Construir la base de datos
    • Manipular la base de datos
    • Compartir la base de datos entre usuarios
  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.En un crawler web que buscaba ciertos expedientes para mi padre, el programa sólo daba el output sin embargo no existía una forma de filtrarlos de forma eficaz
